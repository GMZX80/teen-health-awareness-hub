Accessibility

This site aims to be readable, keyboard-friendly and usable on phones, tablets and desktop screens.

Main guide Design choices Known limits Statement

Aim

Make the guidance easier to use

This is an independent public awareness resource, not a public-sector website. Even so, the site aims to follow good accessibility practice so that families can read, navigate and use it more easily.

Design choices

What the site tries to support

The site uses semantic HTML, headings, lists, descriptive link text, responsive layouts, visible focus-friendly controls where supported by the browser, and plain-language explanations. Pages are intended to work without a login or account.

Some pages include large tables, downloadable-style tools or external documents. These may be harder to use on small screens or with some assistive technologies.

Known limits

What may not be perfect

Some third-party linked content, PDFs, embedded external images, browser print outputs or locally generated notes may not meet the same accessibility standard as the main HTML pages. External websites are outside this site's control.

Statement

Current accessibility position

This statement reflects the site as published on 2026-07-01. The site is a small independent public awareness resource, so it does not claim formal public-sector accessibility certification. The aim is to keep the main HTML pages clear, readable and usable across common devices and assistive technologies.

If a public contact route is added in future, this page can include a way for visitors to report accessibility problems or request information in another format.
